## 👾 Mini-Games

| Command | Category | Description |
| :--- | :--- | :--- |
| `[ /sp-games snake ]` | Single-Player | Play the classic Snake game utilizing reactive Discord buttons. |
| `[ /sp-games 2048 ]` | Single-Player | Slide tiles using interactions to merge matching numbers towards 2048. |
| `[ /sp-games minesweeper ]` | Single-Player | A spoiler-based dynamic minesweeper grid challenge. |
| `[ /mp-games tictactoe ]` | Multi-Player | Challenge another server member to Tic-Tac-Toe directly in chat. |
| `[ /mp-games connect4 ]` | Multi-Player | The strategic disc dropping game, rendered in a Discord embed grid. |
| `[ /guessthenumber ]` | Chat Event | Admins lock a channel to guess a specific number before time runs out. |
| `[ /counting ]` | Chat Event | Setup a channel strictly for consecutive counting—ruin it and restart! |

## 🗣️ Social & Entertainment

| Command | Description |
| :--- | :--- |
| `[ /family marry ]` | Propose to another Discord user to officially link profiles. |
| `[ /family divorce ]` | Annul your current Discord marriage. |
| `[ /fun 8ball ]` | Ask a question and receive a randomized, mystical answer. |
| `[ /fun meme ]` | Pulls a highly randomized meme dynamically straight from Reddit. |
| `[ /hack ]` | Generate a \"fake hacking\" sequence to mock your friends in chat. |

<Tip>
  **Social Growth**: These tools dramatically increase daily server engagement! Using games like Tic-Tac-Toe and Connect4 gives members a reason to stay active inside your community.
</Tip>
